# バトル背景画像ディレクトリ

このフォルダには、バトルモードで使用するステージごとの背景画像を配置します。

## 必要な画像ファイル

以下の背景画像を配置してください（推奨サイズ：1920x1080以上のPNG/JPG）：

### ステージ別背景
- `dialysis_room.png` - Stage 1: 透析室
- `operating_room.png` - Stage 2: 手術室
- `icu_room.png` - Stage 3: ICU
- `laboratory.png` - Stage 4: 検査室
- `blood_room.png` - Stage 5: 血液浄化室
- `pump_room.png` - Stage 6: ポンプ室
- `device_room.png` - Stage 7: 機器管理室
- `demon_room.png` - Stage 8: デーモンの間
- `machine_titan.png` - Stage 9: タイタンの間
- `emperor_hall.png` - Stage 10: 帝王の間
- `abyss.png` - Stage 11+: 闇のICU

## 画像の要件
- 形式：PNG または JPG
- 推奨サイズ：1920x1080 ピクセル以上
- アスペクト比：16:9
- ファイルサイズ：各1MB以下推奨（Web表示のため）

## 画像が未配置の場合
画像が見つからない場合は、グレーのグラデーション背景が自動的に表示されます。
