# バトルモード背景画像配置ディレクトリ

このディレクトリには、バトルモードの各ステージで使用される背景画像を配置します。

## 必要な画像ファイル

- `default.jpg` - デフォルト背景（フォールバック用）
- `bg_dialysis.jpg` - Stage 1: 透析スライム
- `bg_cathlab.jpg` - Stage 2: IABPゴーレム
- `bg_icu.jpg` - Stage 3: ECMOドラゴン
- `bg_ventilator.jpg` - Stage 4: 人工呼吸ケルベロス
- `bg_hybrid_or.jpg` - Stage 5: PCPSリヴァイアサン
- `bg_apheresis.jpg` - Stage 6: 血液浄化フェニックス
- `bg_pacemaker.jpg` - Stage 7: ペースメーカーキング
- `bg_defibrillator.jpg` - Stage 8: 除細動デーモン
- `bg_or.jpg` - Stage 9: 麻酔器タイタン
- `bg_melab_sfx.jpg` - Stage 10: ME機器エンペラー
- `bg_dark_icu.jpg` - Stage 11+: ICUシャドウ

## 画像仕様

- 推奨サイズ: 1920x1080px 以上
- 形式: JPEG または PNG
- 容量: 各1MB以下推奨（パフォーマンスのため）

## 注意事項

画像が見つからない場合は `default.jpg` にフォールバックされます。
最低限 `default.jpg` を配置することを推奨します。
